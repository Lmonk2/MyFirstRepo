HTML + CSS CHEAT SHEET (ADHD MODE)

1. Basic Page Template (Copy-Paste This EVERY Time)

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <!-- your stuff here -->
</body>
</html>

3. CSS Box Model (Never Forget Again)

Margin (outside) → Border → Padding (inside) → Content
Shorthand you’ll use every day: 
padding: 20px;               /* all sides */
margin: 10px 20px;           /* top-bottom  left-right */
padding: 10px 20px 30px 40px; /* top right bottom left (clock) */

5. Colors & Units (Fast Reference)

  • px – pixels (fixed)
  • % – percentage of parent
  • rem – based on root font (use this most!)
  • em – based on parent font
  • vw / vh – % of screen width/height
Color names that actually work: red, orange, purple, teal, crimson, hotpink, limegreen

6. Navigation Bar (Copy-Paste This Entire Block)

<!-- HTML -->
<ul class="nav">
  <li><a href="#">Home</a></li>
  <li><a href="#">About</a></li>
  <li><a href="#">Contact</a></li>
</ul>

<!-- CSS -->
.nav { list-style: none; margin:0; padding:0; background:#333; overflow:hidden; }
.nav li { float:left; }
.nav a { display:block; color:white; text-align:center; padding:14px 20px; text-decoration:none; }
.nav a:hover { background:#111; }
